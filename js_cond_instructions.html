Javascript Conditional Statement Exercises

Exercise 1: Toggle Button Disabled Property

  • Open the files cond_ex1.html and cond_ex1.js in a text editor and load the html file in a browser.
  • Include the Javascript file in the HTML file using a script element.
  • Find two buttons defined in the HTML file: Hello button and Disable/enable hello button.
  • The Hello button has a handler set for the event click. Examine it then click this button to see if what you expected happens.
  • A function called toggleHelloButtonDisabled is set as the handler for the event click on button Disable/enable hello button. Click this button. Nothing should happen because the function currently does nothing.
  • Your task is to find the function toggleHelloButtonDisabled in the Javascript file and define its behaviour, so that it disables button Hello button if enabled and enables it if it is disabled.
  • The element object property that you need to change is called disabled and it should be set to a boolean.
  • A video of the expected behaviour can be found here.

Exercise 2: Calculator Revisited

  • Repeat the calculator exercise from the JS with HTML Inputs lab, making sure that all the operations are handled with a single function. The function should have a parameter called operation, with possible values '+', '-', '*' and '/'.
  • The HTML for this exercise is provided as cond_ex2.html. In it, the handler has been set for the add and subtract buttons but you must add the other event handlers.
  • The Javascript file cond_ex2.js has been included in the HTML but you have to create the file.
